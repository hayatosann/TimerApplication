10,30,60,120,180秒のタイマー設定をプルダウンで選択できて、なおかつメインの画面ではスタート、ストップが行えるように実装。

# 課題

タイマーが終了、つまり時間がきた時のACTIONを設定できるようにはしたいと考えています。
EX)音楽が鳴る、バイブレーションが作動など。
